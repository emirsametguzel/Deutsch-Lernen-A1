Deutsch Lernen A1 - Interactive German Learning Platform

An interactive web-based German language learning platform for A1 level students, featuring real-time progress tracking, vocabulary flashcards, exercises, tests, and a comprehensive teacher dashboard.
🎯 Overview
Deutsch Lernen A1 is a full-featured language learning platform designed for students learning German at the A1 level. Built with HTML, CSS, and JavaScript, it integrates with Firebase for authentication, real-time data synchronization, and cloud storage.
✨ Key Features
For Students
📚 9 Complete Units covering A1 topics:
Personal Information (Informationen zur Person)
School (Die Schule)
Society & Family (Die Gesellschaft)
Daily Life (Tägliches Leben)
Food & Drinks (Essen und Trinken)
Special Days (Besondere Tage)
Leisure Activities (Freizeitaktivitäten)
Shopping (Einkaufen)
Separable Verbs (Trennbare Verben)

🗣️ Interactive Learning Sections
Kommunikation - Dialogues and speaking practice with audio
Wortschatz - Vocabulary flashcards with search, filter, and favorites
Grammatik - Grammar explanations with tables
Übungen - Multiple exercise types (multiple choice, fill-in-blanks, matching, true/false)
Test - Unit tests with instant scoring

🔊 German Text-to-Speech
Native German pronunciation
Fallback to Google Translate TTS when browser voices unavailable
Audio buttons for all vocabulary and example sentences

🏆 Gamification
Points system for completed exercises
12 achievement badges (First Step, Champion, Perfectionist, BABAPİRO, etc.)
Global leaderboard with real-time rankings

👤 Student Profile
Track completed units and test scores
View earned badges
Monitor overall progress

For Teachers
👨‍🏫 Teacher Dashboard
View all students and their progress
Reset student progress
Delete student accounts

📨 Homework Assignment System
Assign homework to entire classes or individual students
Set deadlines
Track student progress per assignment
View class analytics

✍️ Content Editor
Edit unit titles, colors, and descriptions
Modify communication sections and dialogues
Update grammar explanations
Real-time content updates for all users

📚 Unlimited Vocabulary Editor

Add, edit, or delete vocabulary cards
Bulk add multiple cards
Changes sync instantly to all students

🔐 Admin Management
Add/remove admin emails
Admins have full access to teacher features

🛠️ Technical Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)
Backend Services: Firebase (Authentication, Firestore)
TTS: Browser SpeechSynthesis API + PHP proxy fallback
Styling: CSS with dark mode support
Authentication: Email/Password + Google Sign-In

🎮 How to Use
Student Guide
Register with email/password or Google
Complete your profile (name, class, student number)
Navigate units from the home page
Study content in order: Kommunikation → Wortschatz → Grammatik → Übungen → Test
Earn points by completing exercises correctly
Track your progress in the Profile section
Check the Leaderboard to compare with classmates

Wortschatz Features
Flashcards - Click to flip between German and Turkish
Search - Find words in German or Turkish
Filter by unit - Study specific units or all vocabulary
Favorites - Star words for focused practice
Audio - Listen to pronunciation of any word
